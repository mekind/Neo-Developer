# Initial Vault Setup Note

## What was created

A lightweight llm-wiki-style frontend documentation scaffold under `FE/` with:
- root index
- vault config
- scoped `AGENTS.md`
- activity log
- raw/wiki/output separation
- starter indexes and a first overview page

## Why

The repository already had a placeholder `FE/` directory but no documented operating model for how frontend context should be stored. This scaffold creates a predictable place for future docs work.

## Expected Usage

- drop source materials into `raw/`
- synthesize stable understanding into `wiki/topics/`
- maintain lookup pages in `wiki/references/`
- place generated briefs and plans into `output/`
