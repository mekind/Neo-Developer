# FE Bootstrap Note

## Summary

Created the first FE bootstrap definition and scaffold for a lightweight gather-like web direction.

## Chosen baseline

- Vite + TypeScript
- React UI shell
- Canvas world placeholder
- strict TS + ESLint + Prettier + Vitest

## Scope boundary

This is a bootstrap only.
It does not yet implement realtime, WebRTC, auth, backend, or mobile-first support.
